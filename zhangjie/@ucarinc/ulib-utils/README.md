## Utils

通用工具库, 包含:

- cookie cookie 操作
- crypto 简单加解密, base64
- storage 本地(localstorage, sessionstorage)和 cookie 存储管理器

## Install

npm:

```
npm install --save @ucarinc/ulib-utils --registry  https://unpm.10101111.com
```

yarn:

```
yarn install @ucarinc/ulib-utils --registry  https://unpm.10101111.com
```
